Sobre mí

Soy desarrollador backend enfocado en la construcción de sistemas robustos, APIs de alto rendimiento y soluciones que priorizan la estabilidad, la escalabilidad y la claridad del código.

Trabajo principalmente con Python y frameworks modernos como FastAPI, diseñando arquitecturas limpias, integraciones eficientes con servicios externos y flujos de datos confiables para aplicaciones reales. Asi mismo trabajo con bases de datos relacionales y entornos contenedorizados para garantizar la consistencia del sistema.

Me especializo en la arquitectura del servidor, lógica, rendimiento, mantenibilidad y buenas prácticas, asegurando que el flujo de datos sea eficiente para el consumo de interfaces modernas.

  • Rol: Desarrollo backend completo
  • Stack: Python, FastAPI, APIs REST, PDF, manejo de archivos
  • Enfoque: Automatización, escalabilidad y orden del código

Aplicación orientada a la digitalización de formularios administrativos, permitiendo generar documentos estructurados y reducir el uso de papel.

  • Rol: Backend y lógica de generación
  • Stack: Python, plantillas HTML, generación de documentos
  • Enfoque: Estandarización y eficiencia
